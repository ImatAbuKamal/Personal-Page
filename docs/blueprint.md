# **App Name**: PersonaPage

## Core Features:

- Dynamic Content Management: Fetch and display all portfolio sections (Home, About, Gallery, Contact, Footer) dynamically from a Google Apps Script API.
- Carousel Image Display: Showcase a responsive image carousel with captions on the homepage, automatically rotating slides.
- Message Submission Form: Allow visitors to submit messages and an image attachment through a dedicated form, with client-side validation and secure upload to the backend.
- Portfolio Content Refinement Tool: Suggest improvements or rephrasing for project descriptions and 'About Me' text to enhance clarity, impact, and SEO relevance.
- Responsive Navigation & UI: Implement a responsive navigation menu with a hamburger icon for mobile, smooth scrolling to sections, and a 'back-to-top' button.
- Interactive Alerts: Utilize SweetAlert2 for clear and user-friendly success, error, and loading notifications during form submissions and data fetching.

## Style Guidelines:

- Primary color: A sophisticated, slightly muted indigo (#575785) to convey professionalism and depth.
- Background color: A very light, desaturated blue-gray (#EFEFF4) to maintain a clean, open, and inviting aesthetic.
- Accent color: A vibrant yet serene cyan (#59C0E8) to highlight interactive elements and provide visual interest, contrasting with the primary color.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for all content. Note: currently only Google Fonts are supported.
- Integrate Font Awesome 6 for all iconography, ensuring consistency and broad range of symbols. Icons should be clear and functional.
- A single-page, section-based layout optimized for responsiveness across various devices. The contact form is presented in a modal overlay for focused interaction.
- Subtle, smooth animations for UI transitions, carousel slides, and hover effects to enhance user engagement without distracting from content.
